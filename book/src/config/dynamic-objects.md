# Dynamic Objects Config

**File:** `assets/config/dynamic_objects_config.json`
**Struct:** `DynamicObjectsConfig` in `crates/game-dynamic/src/types.rs`

Defines interactive objects in the world — their triggers, actions, state machines, and light effects. Generated by the [glb-parser](../tools/glb-parser.md) tool and manually edited to add behavior.

---

## Light Intensity Scales

Global multipliers applied to imported light intensities. Useful for adjusting Blender-to-Bevy intensity differences.

```json
{
  "light_intensity_scales": {
    "point": 1.0,
    "spot": 1.0,
    "directional": 1.0
  }
}
```

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `point` | f32 | `1.0` | Multiplier for all point light intensities |
| `spot` | f32 | `1.0` | Multiplier for all spot light intensities |
| `directional` | f32 | `1.0` | Multiplier for all directional light intensities |

---

## Nodes

Each key in `nodes` maps to a named object in the dynamic GLB file. The key must match the Blender object name exactly.

```json
{
  "nodes": {
    "torch_light_01": {
      "type": "light",
      "light_info": {
        "light_type": "point",
        "color": [1.0, 0.8, 0.3],
        "intensity": 800.0
      },
      "triggers": [],
      "state": null
    }
  }
}
```

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `type` | string | `"empty"` | Node type: `"mesh"`, `"light"`, `"camera"`, `"empty"` |
| `light_info` | object | `null` | Light metadata (only for `"light"` type nodes) |
| `triggers` | array | `[]` | List of trigger definitions |
| `state` | object | `null` | State machine configuration |

### Light Info

Auto-populated by glb-parser for light nodes.

| Key | Type | Description |
|-----|------|-------------|
| `light_type` | string | `"point"`, `"spot"`, or `"directional"` |
| `color` | [f32; 3] | RGB color (0.0-1.0) |
| `intensity` | f32 | Light intensity (candelas) |

### State

Optional state machine for the object.

```json
{
  "state": {
    "initial": "idle",
    "toggle": false
  }
}
```

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `initial` | string | `"idle"` | Starting state name |
| `toggle` | bool | `false` | If true, triggers alternate between states |

## Triggers

Each trigger fires when a condition is met, executing its list of actions.

```json
{
  "triggers": [
    {
      "type": "playerOnInteract",
      "radius": 3.0,
      "actions": [
        { "type": "toggle_state" }
      ]
    }
  ]
}
```

### Trigger Types

| Type | Parameters | Description |
|------|-----------|-------------|
| `playerOnInteract` | `radius` (f32) | Player presses interact key within radius |
| `playerOnEnter` | `radius` (f32) | Player enters sensor collider |
| `playerOnExit` | `radius` (f32) | Player exits sensor collider |
| `playerOnShoot` | — | Projectile hits the object |
| `onEntityHealth` | `threshold` (f32) | Object health drops below threshold |
| `onEntitySpawn` | — | Fires once when the object spawns |
| `onStateChange` | `state` (string) | Object enters specific state |
| `onTimer` | `interval` (f32) | Fires repeatedly at interval (seconds) |
| `onDelay` | `delay` (f32) | Fires once after delay (seconds) |
| `onTargetStateChange` | `target` (string), `state` (string) | Another named entity enters a specific state |

---

## Actions

Actions execute when their parent trigger fires.

### State Actions

| Type | Parameters | Description |
|------|-----------|-------------|
| `toggle_state` | — | Toggle between states (requires `state.toggle: true`) |
| `set_state` | `state` (string) | Set object to a specific state |
| `enable` | — | Enable the object |
| `disable` | — | Disable the object |
| `collect` | — | Mark as collected (one-time pickup) |

### Visual Actions

| Type | Parameters | Description |
|------|-----------|-------------|
| `display_text` | `text` (string) | Show UI text overlay |
| `hide_text` | — | Hide UI text overlay |
| `set_visibility` | `visible` (bool) | Show/hide the object's visuals |
| `set_material_color` | `color` (string, hex) | Change material color |

### Animation Actions

| Type | Parameters | Description |
|------|-----------|-------------|
| `play_animation` | `animation` (string) | Play a named animation clip |
| `stop_animation` | — | Stop current animation |

### Transform Actions (Tweens)

| Type | Parameters | Description |
|------|-----------|-------------|
| `move_to` | `target` [x, y, z], `duration` (f32) | Tween position over duration (seconds) |
| `rotate_to` | `target` [x, y, z, w], `duration` (f32) | Tween rotation (quaternion) over duration |
| `scale_to` | `target` [x, y, z], `duration` (f32) | Tween scale over duration |

### Light Effect Actions

| Type | Parameters | Description |
|------|-----------|-------------|
| `start_light_effect` | `effect`, `channel`, `min`, `max`, `speed` | Start a light effect (see below) |
| `stop_light_effect` | `channel` (string) | Stop a light effect on a channel |

**Light effect parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `effect` | string | `"flicker"`, `"pulse"`, or `"fixed"` |
| `channel` | string | `"intensity"` or `"color"` |
| `min` | f32 | Minimum value for the effect |
| `max` | f32 | Maximum value for the effect |
| `speed` | f32 | Effect speed (cycles per second) |

### Audio Actions

| Type | Parameters | Description |
|------|-----------|-------------|
| `play_sound` | `sound` (string) | Play an audio clip |

### Timing Actions

| Type | Parameters | Description |
|------|-----------|-------------|
| `delay` | `duration` (f32) | Wait before executing next action (seconds) |

## Examples

### Flickering Torch

```json
{
  "torch_light": {
    "type": "light",
    "light_info": { "light_type": "point", "color": [1.0, 0.8, 0.3], "intensity": 800.0 },
    "triggers": [
      {
        "type": "onEntitySpawn",
        "actions": [
          {
            "type": "start_light_effect",
            "effect": "flicker",
            "channel": "intensity",
            "min": 500.0,
            "max": 1200.0,
            "speed": 6.0
          }
        ]
      }
    ]
  }
}
```

### Interactive Door

```json
{
  "door_01": {
    "type": "mesh",
    "state": { "initial": "closed", "toggle": true },
    "triggers": [
      {
        "type": "playerOnInteract",
        "radius": 3.0,
        "actions": [
          { "type": "toggle_state" }
        ]
      },
      {
        "type": "onStateChange",
        "state": "open",
        "actions": [
          { "type": "rotate_to", "target": [0.0, 0.707, 0.0, 0.707], "duration": 0.5 }
        ]
      },
      {
        "type": "onStateChange",
        "state": "closed",
        "actions": [
          { "type": "rotate_to", "target": [0.0, 0.0, 0.0, 1.0], "duration": 0.5 }
        ]
      }
    ]
  }
}
```

### Proximity Text

```json
{
  "sign_01": {
    "type": "mesh",
    "triggers": [
      {
        "type": "playerOnEnter",
        "radius": 5.0,
        "actions": [
          { "type": "display_text", "text": "Welcome to the arena" }
        ]
      },
      {
        "type": "playerOnExit",
        "radius": 5.0,
        "actions": [
          { "type": "hide_text" }
        ]
      }
    ]
  }
}
```
